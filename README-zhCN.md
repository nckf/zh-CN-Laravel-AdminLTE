---

`中文`

## > 开始！ <
---
> 1 . 安装
````
    cd project

    git clone git@github.com:nckf/zh-CN-Laravel-AdminLTE.git

    mv zh-CN-Laravel-AdminLTE/zh-CN resources/lang/
````

> 2 . 清理
>> 谨慎!
````
    rm -rf zh-CN-Laravel-AdminLTE
````

> 3 . 使用
>> 更改 `config/app.php` 中 `locale` 的值
>>> 'locale' => 'zh-CN'
